# 思维导图编辑器项目说明

## 网站简介
本项目是一个在线的思维导图编辑器，旨在为用户提供一个便捷、高效的思维导图创建和编辑工具。通过直观的界面和丰富的功能，用户可以轻松地创建、编辑和管理思维导图，以辅助思考、规划和组织信息。

## 页面与功能
本项目包含7个主要页面，涵盖了用户从注册登录到创建、编辑和分享思维导图的全过程。主要功能和页面包括：
- **首页**：介绍思维导图编辑器的功能和特点，提供快速入口引导用户开始使用。
- **登录/注册页面**：支持用户注册和登录，保障用户数据的安全性和个性化体验。
- **工作台**：展示用户创建的思维导图列表，支持新建、编辑和删除操作。
- **编辑器页面**：核心功能页面，提供思维导图的创建、编辑功能，包括节点添加、删除、样式调整等。
- **分享页面**：允许用户将创建的思维导图分享给他人，支持多种分享方式。
- **帮助/教程页面**：提供使用指南和教程，帮助新用户快速上手。
- **关于页面**：介绍项目背景、团队信息等。

## 多 AI 协作与验收
本项目采用多AI协作的方式进行开发，不同的AI模型负责不同的任务，如前端界面设计、后端逻辑实现、功能测试等。通过明确的任务划分和接口定义，确保了各部分之间的协同工作。项目的验收通过自动化测试和人工评审相结合的方式进行，确保了项目的质量和稳定性。
