思维导图编辑器

使用帮助

快速入门教程

  1. 打开编辑器：访问 index.html，页面加载后即可看到空白画布。
  2. 添加节点：点击左侧工具栏的 “+ 节点” 按钮，或按快捷键 N，在画布中点击即可创建节点。
  3. 编辑节点：选中节点后，右侧属性面板会显示该节点的文字、颜色、形状等属性，直接修改即可生效。
  4. 连接节点：选中一个节点，拖动其右侧出现的小圆点（连接点）到另一个节点，即可创建连接线。
  5. 移动与删除：拖动节点可移动位置；选中节点后按 Delete 或点工具栏 “- 删除” 按钮删除。
  6. 保存：编辑器会自动将当前思维导图保存到浏览器的 IndexedDB，刷新页面或下次打开时会恢复。
  7. 导出：完成编辑后，可访问 导出页面 导出为 PNG、JSON 或 SVG。

常用快捷键

快捷键 功能
N添加新节点
Delete / Backspace删除选中节点
Ctrl + Z撤销
Ctrl + Y重做
Ctrl + S手动保存（亦会自动保存）
Ctrl + E导出思维导图（跳转至导出页面）
Ctrl + F在帮助页面中搜索（仅本页）
Esc取消选中或关闭属性面板

常见问题解答（FAQ）

如何切换主题或更改默认字体？

进入 设置页面，可在此修改编辑器的主题颜色、默认字体以及自动保存间隔。

是否支持多人协作？如何邀请他人？

在编辑器右上角点击 “共享” 按钮，生成一个可邀请的链接，发送给他人后即可实时协作编辑同一份思维导图。
